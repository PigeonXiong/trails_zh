# Introduction 引言

**欢迎来到《英雄传说：命运之轨迹》。**

这款游戏是您踏入这个世界的入口——一个由政治博弈、古代科技，以及建立在世代冲突之上的脆弱和平所构筑的世界。在这个世界里，导力革命改变了一切，从国家发动战争的方式到普通人维持日常生活的方式，概莫能外。而如今，变革再度降临。

这款游戏邀请你成为那些能够塑造未来的少数人之一。作为玩家，你将扮演一个身处愈演愈烈之动荡中心的角色。无论你是一位漂泊的游击士、一位冉冉升起的新星贵族，还是一位寻求人生意义的退伍老兵，你的故事都将跨越国境、信仰与战场展开。

我们构建这个系统，是为了让你有充分的自由去发掘你角色的本质，而不仅仅是他们的能力。你所做的每一次行动检定、你选择的每一个天赋、你抓住的每一次机会，都将引领他们更加接近自己的命运。这条道路不会平坦。但传奇之路，从来如此。

感谢你选择在这个世界游玩。无论你的战役是绵延数年，还是仅仅留下几个难忘的片段，我们都希望《命运之轨迹》能成为承载你毕生难忘之故事的舞台。前路漫漫，同伴寥寥，赌注高昂。但这也正是你在此处的原因。

《命运之轨迹》的世界存在于进步与崩溃之间的微妙平衡中。埃雷波尼亚、克洛斯贝尔以及塞姆利亚大陆上星罗棋布的诸国，通过外交、阴谋与武力行使着各自的权力。但在这些政治角力之下，古老的力量正在涌动；被遗忘的遗物、失落的文明以及沉寂已久的魔法系统，逐年苏醒。

这是一个由转型所定义的时代。旧秩序分崩离析。新意识形态生根发芽。曾经无权无势的人开始奋起，而那些掌权者必须直面“控制权正从他们指缝间溜走”的现实。导力时代带来了创新的浪潮，但这创新也伴随着代价——技术依赖、经济差距以及传统的消逝。

魔法与机械勉强共存。奥术与科学，神圣与世俗，始终处于紧张的对抗之中。有些人试图调和它们。另一些人则旨在利用一方支配另一方。在这种变幻莫测的混合体中，你们的角色将成为变革的推动者，无论他们是否自愿。

《命运之轨迹》并非关乎拯救世界。它关乎的是“活在一个值得拯救的世界里”究竟意味着什么。它关乎在变幻的忠诚面前，关于妥协、信念与身份认同的探讨。遗产、责任与个人成长的主题无处不在。

这是一个历史沉重地压在当下肩头的世界。你踏上的轨迹并非前所未有。前人已在此行走过。但你的选择将塑造这些道路通往何方。

你将扮演那些必须在**塞姆利亚**生活下去的人。他们并非生来就是英雄。大部分人甚至不认为自己有何特殊之处。一个从东方前线逃离的士兵，一个躲避债务、在街头摸爬滚打的精明信使，一个因质疑圣典而被驱逐的修女——他们都起步于大陆的不同角落。不同的故事，不同的伤痕。

将他们聚在一起的并非命运。或许是偷来的铁路通行证，或许是一个共同的敌人，又或许是在错误的地点遇上了正确的时间。他们未曾计划成为更宏大事业的一部分。但当导力网络在整个城市陷入黑暗时，或者当王都废墟下沉寂了数个世纪的遗迹突然亮起时，他们就在现场。当这个世界需要有人挺身而出之时，他们没有袖手旁观。

他们之间的羁绊是在应对危机的过程中锻造的。不是因为友谊，也不是忠诚，至少一开始不是，而是因为他们意识到，只有他们看清了全部的画面。他们头顶上的那些权力不会行动。或许是不能行动。

让他们与众不同之处并非天赋，尽管有些人确实天赋异禀。那是担当。当大多数人面对古代科技或神迹现象转身逃跑时，这少数几个人却朝着它们走去。当体制失效时，他们会追问如何才能改变它。他们承受着这样一种认知所带来的重负：历史正在重演，而这一次，他们有角色要扮演。

他们会有争执。他们会失败。他们会战斗。

但他们将一起行动。

因为如果他们不做，就没人会做了。
