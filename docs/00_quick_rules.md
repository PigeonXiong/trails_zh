# Quick Rules 快速规则

## 基础

### 动作检定 Action Rolls

当任何行动可能失败或存在对抗时，你进行一次**动作检定**。

!!! formula ""
    动作检定：1d20 + 加值骰


### 成功总和 Victory Sum (VS)

动作检定的目的是达到或超过一个成功总和（VS）。示例 VS：

| 难度层级 | VS  |
| -------- | --- |
| 微弱     | 5   |
| 低劣     | 10  |
| 普通     | 15  |
| 英雄     | 30  |
| 史诗     | 50  |
| 传说     | 70  |
| 神话     | 110 |
| 超越     | 150 |

### 加值骰 Bonus Dice

每颗加值骰一般是 ``d10``。你的**属性**值和**技能**会为动作检定增加一定数量的加值骰。

### 骰等阶 Dice Ranks

**充能**和**挑战**会分别提高或降低加值骰的骰面，这称作**骰等阶**。

| 骰等阶 | 骰面 |
| ------ | ---- |
| 微弱   | D4   |
| 低劣   | D6   |
| 普通   | D8   |
| 英雄   | D10  |
| 史诗   | D12  |
| 传说   | D20  |
| 神话   | D100 |

### 机会 Chances

目标VS增加 `+15`，但给成功后的效果或**效果检定**（如判断伤害或恢复的骰子）增加 1 充能。你在任何检定上最多可以选择承受 3 机会，且机会带来的充能不能让最终效果超过传说（也不能让效果检定的加值骰超过`d20`）。

### 挑战 Challenges

将动作检定的加值骰降低 1 个等阶。如 d10 变为 d8。

### 充能 Charges

将**效果**(Effect)或**效果检定**中的所有**加值骰**提高 1 个等阶，例如提高制作物品的品质、或提高伤害检定的加值骰的骰面。

!!! note "译注"
    挑战降低检定成功率、充能则提高成功后的效果（或效果检定）。而机会则是主动提高VS来换取充能。

## 技能与属性值 Skills & Ability Scores

技能使用情境上相关的某种属性值。技能动作见下：

- **运动**：奔跑、攀爬、游泳、举重或跳跃。
- **警觉**：发现危险、注意细节或进行调查。
- **驯兽**：驯服、训练或骑乘动物和坐骑。
- **制作**：锻造、裁缝、木工、宝石切割或建筑。
- **防御**：针对武器类攻击的闪避、招架或格挡反应。
- **娱乐**：音乐、舞蹈、表演或演出，用于激励、分散注意力或施加影响。
- **重武器**：使用双手武器（大剑、斧、锤、杖和长柄武器）攻击。
- **诡术**：开锁、戏法、偷窃或欺骗。
- **轻武器**：使用单手武器（剑、刀、斧）攻击。
- **医药**：治疗伤势、外科手术、诊断或药理学。
- **神秘学**：施放法术、举行仪式、阅读奥术符号或制作药剂。
- **导力魔法**：使用导力器和结晶回路施放魔法、控制装置或强化盟友。
- **驾驶**：操作载具、特技驾驶或空中机动。
- **远程武器**：使用弓、投掷武器或火器攻击。
- **抵抗**：克服物理、精神或超自然的袭击和状态。
- **社交**：魅力、说服、欺骗、威吓或洞察他人。
- **潜行**：隐藏、无声移动或避免被发现。
- **学问**：学术知识、法律、历史、语言和密码学。
- **生存**：追踪、狩猎、觅食或在荒野中求生。
- **科技**：操作机器、修理系统、机器人、计算机，以及进行实验室科学（化学、生物学、物理学）。
- **徒手**：使用武术、斗殴或天生武器进行打击。

## 战斗规则 Combat Rules

### 先攻 Initiative
先攻为 1d20，在遭遇开始时投掷。回合按检定结果从高到低的顺序进行。平手时，拥有最多速度加值骰的战斗者先行动。

### 动作 Actions
所有生物默认拥有两个动作。在你的回合，你可以使用动作来移动、攻击或使用能力。

### 攻击检定 Attack Rolls
使用武器攻击时，你进行一次技能检定 vs 目标的**防护**(Guard)。防护默认为 15。

### 施法检定 Casting Rolls
以对手为目标使用法术或魔法能力时，你进行一次技能检定 vs 目标的**抗护**(Ward)。抗护默认为 15。

!!! formula ""
    攻击检定：1d20 + 技能加值骰 + 敏捷(Agility)加值骰

    施法检定：1d20 + 技能加值骰 + 灵魂(Soul)加值骰

### 尝试防御 Defending
你可以通过一次成功的**防御检定**(Defense Check)来尝试闪避对你发动的**攻击**(Attack)。防御消耗 5 点**耐力**(Vigor)。

### 尝试抵抗 Resisting
你可以尝试克服影响你精神、身体或灵魂的**袭击**(Assault)，进行一次**抵抗检定**(Resistance Check)。抵抗消耗 5 点**魔力**(Mana)。

!!! formula ""
    防御检定：1d20 + 防御加值骰 + 速度(Speed)加值骰

    抵抗检定：1d20 + 抵抗加值骰 + 目标属性加值骰

### 恢复阶段 Recovery Phase
在每轮结束时，每人每有 5 点体魄(Stamina)便恢复 1 点耐力。如果你有生效中的效果，你可以选择消耗 5 点魔力来抵抗其中之一。
