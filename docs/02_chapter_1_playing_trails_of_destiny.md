# 第一章: 游玩《命运之轨迹》

本游戏采用一套名为钷系统(Promethium System)的简易规则。

你将扮演一名由你自行创造的英雄，探索一个充满危险与冒险的世界，而这些规则将助你解决竞争与冲突，并推进角色的成长。其中一名玩家将担任Game Master，负责裁断规则。Game Master将描述你的遭遇，并扮演你所面对的造物、角色与敌人。

你们将共同打造一段自由形态的叙事，并通过你的行动及其后果来塑造故事。

## 设定角色

作为玩家，你将化身为一名由你自行构思的角色。希望这个角色会是风趣、独特，且远不止于描述其的数据与规则。与你一同游玩的其他玩家，也各有其角色，共同组成了这场冒险的阵容。你必须运用你的想象力，在脑海中构建场景，并扮演你所塑造的这个角色。这正是所有角色扮演游戏的核心所在。

## 搭建舞台

你将想象GM所描述的场景，并在其中构想你的角色形象。你会通过你的角色，以你角色的人格化身来观察这个世界并采取行动。GM会告诉你身处何处、看到什么以及局势如何。你的职责就是以角色的身份，也就是作为玩家，根据这些信息展开行动。

## 行动！

最困难的部分在于，决定你以角色身份要采取何种行动。是从悬崖上一跃而下，正面攻击守卫？还是静待时机，从阴影中发动奇袭？角色扮演游戏最棒的一点，就是你扮演的角色几乎可以做到任何事。你的选择或行动没有限制，尽管基于你所扮演角色的能力，你的所知所为会受到一定制约。

当选择角色要采取的行动时，请记住你是在通过角色的眼睛而非自己的眼睛体验事件。你的角色，因此也就是你，应当以符合其观点、知识和个性的方式来行动。

## 使用骰子

钷系统的核心是几个简单的概念：用于解决对抗（例如在战斗中攻击或防御）的**动作检定**；用于决定结果（包括你用武器击中敌人后造成多少伤害）的**效果检定**；以及用于确定你在进行动作或效果检定时使用何种骰子的**骰等阶**。

在本文中，你会看到骰子类型缩写为 d20、d12、d10、d8、d6 和 d4。右边的数字是骰子的面数。当你看到类似 3d8 或 1d20 的数字时，这是你需要投掷的骰子数量与类型。例如，当你看到 3d8 时，这意味着三个八面骰子。

### 动作检定 Action Rolls

!!! formula ""
    动作检定： 1d20 + 属性加值骰 + 技能加值骰 对抗 成功总和VS

当你想采取一个动作，比如进行攻击或拆除炸弹时，你进行一次动作检定，即 1d20 加上任意个加值骰。你的目标是掷投结果大于等于 GM 设定的**成功总和**（VS）。如果你的动作影响另一个生物，它通常有权进行一次反应，例如使用防御来闪避或偏转攻击，或使用抵抗来克服法术或效果。

!!! note "译注"
    快速规则中，检定需要meet or exceed VS，而此处写的是higher than VS。姑且认为是大于等于。

动作检定的难度由动作的性质和涉及的风险决定。许多动作的 VS 值是固定的，以减少模糊性，但你的 GM 或特定规则可能会决定或影响你对抗的 VS。作为玩家，你也有能通过承受**机会**来为你的检定加注。


!!! note ""
    艾丝蒂尔：你不需要立刻学会所有规则！说到掷骰子，只要记住：你不只加一些加值数字，你加的是一个个加值骰！

### 成功总和 Victory Sum (VS)

通常，《命运之轨迹》中的成功总和由叙事条件（风险和动作类型）明确定义，并使用与骰子和品质相同的等阶：从微弱到超越。

| 难度层级 | VS  | 动作性质与风险                                                               |
| -------- | --- | ---------------------------------------------------------------------------- |
| 微弱     | 5   | 风险可忽略的例行动作；琐碎的任务。                                           |
| 低劣     | 10  | 风险较小的例行动作；简单日常。                                               |
| 普通     | 15  | 有风险的例行动作；有压力的任务；风险较小的英雄壮举；雄心勃勃但能掌控。       |
| 英雄     | 30  | 生死攸关的例行动作；极端压力下的关键任务；有风险的英雄壮举；挑战极限的难题。 |
| 史诗     | 50  | 生死攸关的英雄壮举；具有重大风险或影响的非凡伟业。                           |
| 传说     | 70  | 涉及近乎不可能的条件或压倒性风险的例外英雄壮举。                             |
| 神话     | 110 | 超越性的动作；超越正常凡人极限的任务，定义非凡英雄主义或绝境的时刻。         |
| 超越     | 150 | 近乎奇迹的伟业、改变现实的事件、永久影响故事或世界的传奇成就。               |

### 效果 Effect

大多数动作会产生**效果**，例如伤害或技能的结果。在这些结果可变的情况下，你将进行一次**效果检定**(Effect Roll)。

### 加值骰 Bonus Dice

每当你在检定中获得加值(Bonus)时，无论是**动作检定**还是**效果检定**，你都会获得额外的骰子。你的属性、技能和天赋会在各种动作和效果检定上给予加值骰，情境也可能为你提供加值骰。加值骰通常是英雄等阶，但负伤、受伤和其他状态可能会降低它们的骰等阶。

### 骰等阶 Dice Ranks

所有骰子都有一个骰等阶。你在任何动作中投掷的动作骰(Action Roll)都是传说等阶，即`d20`。加值骰则是英雄等阶，即`d10`。

| 骰等阶 | 骰面 |
| ------ | ---- |
| 微弱   | D4   |
| 低劣   | D6   |
| 普通   | D8   |
| 英雄   | D10  |
| 史诗   | D12  |
| 传说   | D20  |
| 神话   | D100 |

### 骰子爆发 Bursting Critical Rolls

当你在**任何加值骰**上掷出最大值时，这些骰子**爆发**(Burst)，你重新投掷其中一个骰子，并将其结果额外加到总和上。这在动作和效果检定中都会发生。

!!! note "译注"
    此处原文是当多个骰子爆发时，重掷其中一个(roll one of these dice again)。但根据discord询问，多个骰子爆发时，全部可以重投并加入结果。

!!! note ""
    艾丝蒂尔：加值骰通常是d10，所以如果你在加值骰上掷出10，它就会爆发，你可以再掷一次并把结果加到你的检定上。讲真，爆发的次数也没上限！

### 机会 Chances

你经常会发现自己需要投掷动作骰和额外2~3个加值骰。在这种情况下，VS 20可能看起来微不足道。作为玩家，你可以选择拿自己的成功来冒险，通过在一次动作上最多承担三次**机会**(Chance)来玩一场孤注一掷的游戏。你每承担一次机会，该动作的VS就会增加15，但你获得 1 **充能**，这会提高结果或效果检定的等阶。

例如，在攻击一个敌人时，你可以尝试一击斩首，承担2次机会，将伤害的加值骰从英雄等阶提升至传说等阶。但这项任务会困难得多，VS增加30。

**限制为三**：无论你有多少骰子可用，在任何一次动作检定上，你最多只能承担三次机会，即 VS+45 但增加 3 充能。

虽然每个核心技能树都包含了可能的机会列表，但充能对结果产生何种影响通常由GM决定。在战斗中，承担机会可以提高伤害加值骰的等阶、允许你造成非致命伤害或进行瞄准攻击。但对于更具叙事性的动作，如**警觉**(Awareness)，GM应当给你提供额外的或更高质量的信息。

机会是在你的回合中、由你主导发生的事情，因此你不能在**反应**或**效果检定**上承担机会。承担机会是指你在选择进行一个动作时，押上一切去博取更大回报。

**传说机会**：无论你在一次动作检定上承担了多少次机会，你都不能将效果推到超过传说等阶（也不能将效果检定的加值骰推到超过`d20`）。这包括伤害骰、物品品质以及任何其他效果。


### 充能与挑战 Charge & Challenges

在某些情况下，临时情况会改变动作的结果，这反映在充能和挑战上。

**充能**(Charge)：一次充能意味着你应用于某个动作或效果检定（如伤害或治疗量）的加值骰等阶提升。例如，如果你在攻击检定上承担了1次机会，将命中所需的VS提高+15，你就会获得1次充能应用于伤害检定的加值骰，将它们的等阶从英雄(`d10`)提升至史诗(`d12`)。充能也能影响你制造的物品的等阶或你能获得的信息，最终将其品质提高一个等阶。

**挑战**(Challenge)：当你面临妨碍你能力的局面时，你可能会受到挑战。每个挑战会使你动作的加值骰等阶降低一阶。最常见的挑战形式是负伤，它会影响你进行的所有动作。通常，挑战是具体且有情境性的。GM可以裁定，在水下用近战武器攻击会为你的攻击和伤害检定带来2个挑战，但射箭则不会。

### 向下取整 Round Down

有许多情况下，数值、效果或数字会进行乘除运算。遇奇数时，一般向下取整。

!!! note "译注"
    并非翻错，此处的取整的确是写遇到奇数时，向下取整至最近的偶数。


## 命定：拥抱你的时刻

在《命运之轨迹》的世界中，命运在荣耀、抗争与信念的时刻激荡。当角色战胜万难时，他们可能获得**命定**(Destined)，进入一种高度清醒与决意的状态。无论是凭借纯粹的技术还是命运的转折，成为命定意味着你的角色正站在伟大的门槛上。

### 进入命定 Become Destined

当以下情况之一发生时，你进入命定：

* 你在动作骰中掷出天然20。
* 你在一次检定中承受3次机会并成功。
* 你在单次检定中爆发了3个加值骰。
* 你执行了一个GM认为在叙事上强大、具有象征意义或英雄气概的行动。

当你进入命定时，你可以保留此状态直到使用它。一个角色一次只能拥有一个命定状态；该状态不可叠加。最后，你不能在命定解放的同一回合内进入命定，也就是说，你不能在命定解放的同一个动作、反应或干扰动作上再次进入命定。

### 命定解放 Unfolding your Destiny

当你处于命定状态时，你可以选择解放你的命运，消耗此状态为关键时刻注入非凡的努力。你可以在下列条件下随时这样做。

你可以**命定解放**来做以下事情之一：

* 在你的回合，以一个动作激活你的S战技，仍需满足其耐力或魔力消耗
* 为一次动作或反应检定增加 +3 加值骰
* 让承受机会带来的充能可以将一次行动检定的结果提升至传说等阶以上
* 在你的回合，以一个动作恢复1/4的最大耐久、耐力**或**(or)魔力
* 立即在回合外执行1个动作，只要不是其他玩家的行动回合中

命定不仅仅是奖励：它是你的角色在关键时刻塑造自己命运的机会。无论是时机完美的攻击、英雄般的力量爆发，还是传说技艺的施展，成为命定代表着你向传说迈进。当你的时刻来临时，要让它发挥作用。

## 优先与先例

当遇到疑问时，最好的先例原则是：情景规则优先于全局标准。每项规则都由其适用的情景界定。一把长剑是一件普通武器，造成普通伤害骰，但当它作为英雄武器时，则造成英雄伤害骰。

在每种情景中，规则都可能存在例外或调整，这是完全正常的。规则的设计就是为了容纳这些情景的存在，而非阻止它们或消除任何例外情况。
