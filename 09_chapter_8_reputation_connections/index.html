    第八章 — 声望与人脉

    塞姆利亚拥有复杂而微妙的社会政治格局。 在冒险过程中，你会找到盟友、敌人，以及那些或许两者兼而有之的人物。这些是角色在旅途中与有影响力的人物、盟友和组织建立的联系和网络。你可以利用这些人脉获取关键信息、获得稀有物品、确保安全通行，或在关键时刻索要人情。加入强大的组织能让你学习精英天赋或秘密战技，而与政治势力的联系甚至可以赋予你一定的职权。 在实际游戏中，玩家将通过角色扮演互动和完成符合联络人利益或需求的任务来建立和培养关系。这些关系的强度以1到5的等阶来衡量，它决定了联络人愿意提供的支持程度和资源多寡、在组织中的影响力或受到的尊重，或者角色能够行使的职权或力量的大小。

    声望

    在塞姆利亚大陆，你的声望值反映了你在各个地区和势力中的知名度和影响力。如同属性值一样，你的声望会在与NPC进行某些互动和行动时为你提供加值骰。然而，你的声望也会影响你获取独家资源的权限。 你可以通过显著事迹、成功完成的任务以及积极的互动来提升声望值。随着声望的增长，机会和收益也会随之增加，从而提升你在塞姆利亚错综复杂的社会网络中的地位和影响力。

    起始声望

    你的起始声望值为10，再加上你天赋带来的任何加值。

    使用声望

    当你尝试利用人脉或行使你的影响力与职权时，就需要用到声望。在这种情况下，你可能需要进行声望检定。这是一次动作检定，你将投掷1d20，并加上你的声望加值骰（如果有的话）。 当你试图利用联络人的资源时，就需要进行声望检定。联络人可以为你提供信息、货物、服务，甚至安排工作。你所属的组织可以提供人力、资源，甚至训练。拥有职权，你或许能够指挥部队、征用重要设备，甚至影响政治和国家事务。联络人的类别和等阶决定了获得此类帮助所需的声望检定VS。 除了主动使用声望之外，基于你的声望值，你还会因所属和职权获得特殊好处。这可能包括解锁精英技能树、精英天赋池，以及学习特殊战技的机会。 然而，所有人脉都伴随着责任。联络人会要求你提供相应的帮助，或为其服务索要报酬。作为组织的一员或行使职权，你需要完成工作，服从上级，甚至可能需要为事业冒生命危险。某些人脉关系也伴随着风险。反复利用犯罪或地下世界的人脉可能会使你陷入法律纠纷，甚至招致敌对组织的威胁。滥用职权或军事资源则可能引发国际事件。 机会与声望 作为一次动作检定，你可以在声望检定上把握机会。在声望检定中把握机会，可以让你暂时将一个联络人的品质提升一个等阶，尽管这只限于一次效果。然而，这样冒险行事会损害你的声望。当你在声望检定中把握机会时，你的声望值将减少1点。与大多数动作检定不同，你一次只能在声望检定中把握一次机会。

    构建声望

    声望基于你的所作所为。它无法轻易买卖，而你的角色扮演和行动将决定你的声望在整个生涯中的变化。即便如此，随着你推进生涯，有若干种方式可以提升你的声望值。

    获得等级。你的基础声望值为10，每提升一级，声望值便+1。但这并不保证你的声望总是等于10加上你的等级。频繁的失败与不断要求糟糕的帮忙会令声望跌落到起始值以下，但随着你完成能获得经验值的壮举，声望终将无可避免地提升。

    成就与事迹。当你完成了某事 — 尤其引人注目、戏剧性或出类拔萃之事时，GM可能会给予你1点声望。若你在为人脉执行任务或工作时表现出色，则更是如此。一次成就是否提升你的声望，完全由GM裁定，但通常当你完成联络人或人脉提供的重要任务时，你的当前声望值应当+1。事迹，由于格外引人瞩目，会使你的声望值+3，除非该事迹另有说明。

    为人帮忙。最后，当有人请你帮忙 — 比如服务或物品 — 你可以选择帮忙，从而有可能提升声望。只有重要的帮忙才有效；借给NPC几米拉并不算数，但将一家人偷运过埃雷波尼亚的边境检查站则毫无疑问。在许多情况下，帮忙和工作界限模糊，区别在于帮忙通常不会获得报酬。

    失去声望

    就像建立声望一样，你的行为也可能损害声望。当你的声望受损时，声望值将降低1点或更多。有几件事可能损害你的声望值：

    请求帮忙。特别重大的帮忙可能会损害你的声望。每当你向联络人或人脉提出请求，并在声望检定中赌上机会时，你的声望便会受损，声望值-1。

    任务与使命失败。如果你通过自己的人脉获得了工作或任务，你就被寄予成功的期望，因为失败会损害你的声望。若作为你持续所属的一部分而交给你的工作或任务彻底失败，你将失去1点声望。同样，任务的部分失败甚至平淡无奇的成功，可能意味着声望不变 — 倘若原本的成功本可使你获得声望值提升的话。

    失信行为。最后，当你未能满足人脉的需求，或以其他方式苛待他们时，你可能损害自己的声望。GM会判定你的行动是否对声望造成损害，但大体上，当你未能支持自己的人脉，未能偿还人情或债务，或者做出了违背你所属组织利益的行为时，你的声望就会受损。在这种情况下，GM可能因你的行为决定将你的声望值降低1点。

    人脉

    在你的职业生涯中，你会培育三类不同的人脉：联络人、所属与职权。每个人脉都有不同用途。联络人可以作为个人为你提供帮助，所属让你获取资源与支援，职权则让你能够行使法治之力。

    理解人脉

    每项人脉都关联着一个类别、品质与等阶。这些界定了你的人脉范围。使用人脉需要付出代价：联络人期待金钱或人情，组织期待忠诚与任务成功，而权力机构则期待你为维系支撑你权力的秩序而工作。

    范畴 人脉分为以下分类范畴：企业、犯罪、政府、佣兵、宗教、神秘学及地下世界。每个范畴代表一个特定的影响力领域。

    企业。拥有这个范畴的人脉，你与大陆上的主要经济力量搭上了线，比如爱普斯坦财团或Verne Company。企业生活可能残酷无情，联络人或许为提升地位与权力而不择手段，而所属关系往往要求交出过硬的成果。

    犯罪。此类人脉极度危险，与黑市及有组织犯罪直接相关，包括Heiyue。犯罪人脉能够接触任何数量的服务或物资，但伴随着重大的法律与道德风险。犯罪人脉的语言是金钱。

    政府。此类人脉属于塞姆利亚某一民族国家现行政府的军事、政治与法律权力领域。政府人脉有法治支撑，但这也意味着会引发政治影响。政府成员身份被归为职权。

    佣兵。此类人脉属于塞姆利亚军事工业复合体的一部分，但在国家常备军的监督之外，包括游击士协会或Jaegers这类团体。佣兵人脉受其公会权力与规章的约束，却携带着可观的武装力量。

    宗教。七耀教会掌控着塞姆利亚大陆的大部分地区，包括其分支与下属教派。他们不仅认可民族国家的权力，还左右着历史的进程。与教会或其附属组织的人脉关系，伴随着道德与伦理上的期待。

    地下世界。这类人脉本质上属于半犯罪性质，在文明的阴影与法律的边缘运作。暗影特工、秘密结社与神秘的权力掮客；地下世界人脉与犯罪人脉的不同，在于它们的目标。地下世界人脉往往怀有崇高而危险的目的。

    神秘学。古老仪式的践行者、灵性力量的阐释者、超自然领域的引路人。他们可能担任灵媒、秘法师、异端学者，或是魔女眷族、噬身之蛇甚至七耀教会内部激进派系等隐秘组织的入门者。这一范畴涵盖正统与非法两种魔法传承，让你得以接触无法通过科学或军事手段获取的传说、至宝与秘传知识。

    等阶 你的联络人愿意为你做到什么地步，你能调遣组织中的多少力量，或者你拥有的政治法律权限有多大，都取决于你人脉的等阶。你可以通过多种方式提升人脉的等阶：为该人脉执行任务，每等级最多进行一次成功的声望检定，以及完成事迹。所有人脉都有1到5的等阶，大多数人脉的初始等阶为1。

    等阶1 人脉 * 联络人：公事公办。 在此等阶，你的联络人认识你并且愿意与你做生意，甚至可能提供一些便利，但也仅限于此。 * 所属：初级成员。 在此等阶，你在组织中是一个新人，虽然可以使用他们的训练资源，但你应当服从命令，而非下达命令。 * 职权：小官员。 在此等阶，你只不过是个小官员，虽然你可能在家乡有些权力，但对于你的国家和整个塞姆利亚大陆来说，你无足轻重。

    等阶2 人脉 * 联络人：信赖的盟友。 在此等阶，你的联络人视你为可以依赖的人。他们会为你提供更好的机会，包括风险更高但回报更大的帮助。这个联络人可能会分享更敏感的信息，或者不遗余力地协助你，但他们会期望你的忠诚和偶尔的回报。 * 所属：受认可的成员。 你已在组织中证明了自己的价值，获得了一定的责任。你现在可以领导小型任务，训练新人，并获取更先进的资源或技能。然而，你仍然需要服从高阶成员的命令。 * 职权：区域官员。 作为区域官员，你在特定地区或地带行使权力。你可以影响地方政策，指挥小股部队，并调用有限的国家资源。但是，你的决策仍会受到高层严密监督，且你的权力不会超出你的直接管辖范围。

    等阶3 人脉 * 联络人：密友。 此联络人现在足够信任你，愿意分享至关重要的、甚至是危险的信息和极难获得的人情。在此等阶，他们可能会不顾自身职位来帮助你，但他们会期望你高度忠诚，并可能希望你更深入地卷入他们的事务。 * 所属：资深成员。 作为资深成员，你已对组织的方向有了一定的影响力。你能够领导更重要的任务，并获得精英训练或装备。你的意见在内部政治中分量很重，经常被征求意见，尽管你仍需听命于更高的权威。 * 职权：行省统治者。 你已被提拔到一个影响力跨越一个行省或多个地区的职位。你可以指挥可观的力量，通过指示，并显著地影响区域事务。然而，你的行动开始带来更多的政治后果，敌对派系可能开始将你视为威胁。

    等阶4 人脉 * 联络人：关键伙伴。 你的联络人视你为其关系网中的关键人物。他们愿意不遗余力地支持你，包括调动大量资源，冒着个人安全风险，或牵涉强大的第三方。在这个级别，这种关系很可能是互惠的，双方都高度依赖对方。 * 所属：指挥官。 你在组织中担任指挥职位，监督大型团队或项目。你对组织战略有重大影响力，你的等阶使你能够获取高级信息、独家训练和大量资源。在此等阶，你可以开始拥有一定的自主权，向高层决策发起挑战。 * 职权：国家行政官。 作为高级官员，你的影响力遍及全国。你可以指挥大军，影响国家政策，并请求顶级资源。你的角色伴随着巨大的风险，因为来自政府内外的敌人都可能试图动摇你的地位。

    等阶5 人脉 * 联络人：信赖的顾问。 你已与这位联络人的关系到达顶峰。他们信任你到愿意分享最隐秘的秘密，并会为你的成功押上一切。你对他们有绝对的影响力，但这种程度的信任意味着你已深陷他们的事务之中，他们的敌人也会变成你的敌人。 * 所属：执行官。 你是组织的最高领导者之一，可能仅次于其创始人或最高议会。你对资源和人员拥有近乎完全的控制权，你的决策塑造了组织的方向。虽然你享受着巨大影响力带来的好处，但这也意味着你会成为敌对派系的标靶。 * 职权：最高权力者。 你现在对国家的某一权力部门拥有近乎绝对的权威。你的决策影响着国家政策、军事行动和国际关系。在此等阶，你可以重塑政治格局，但你所做的每一步都会受到外国势力和国内政敌的密切关注。

    品质 归根结底，并非所有人脉都是平等的。人脉的品质代表了该人脉权力和影响力的范围。一个普通品质的佣兵联络人可能不过是一个小队指挥官，能提供有限的情报和资源，而一个史诗品质的佣兵联络人很可能是一位高级军官，可以随意调遣士兵和分配任务。人脉的品质决定了为达成机制效果（如工作报酬或研究检定的加值）而投出的任何骰子。联络人的品质绝不能超过其所属组织或其拥有职权的品质。

    联络人 * 普通品质的联络人。 这种联络人拥有有限的权力或影响力。可以想象成克洛斯贝尔警察局的一名探员，或者游击士协会的一名低阶成员。他们能提供不错的情报、小任务和一些资源。他们在本地有很好的关系网，但其触手仍局限于较小、影响力较低的圈子。 * 英雄品质的联络人。 英雄品质的联络人在其领域内地位稳固，例如一位知名的游击士或爱普斯坦财团的中层成员。他们能获得有价值的物品、可执行的情报，或重大的人情，如安全通过危险区域的通行权。这些联络人对自己的地位充满信心，并愿意为你采取冒险行动。 * 史诗品质的联络人。 这种联络人是其领域中最具影响力的人物之一。他们可能是噬身之蛇组织中的高层人物，或像莱恩福尔特这样的巨头企业中的高级官员。他们能够获取稀有技术、军用级装备和绝密情报。然而，这些高风险的人脉伴随着巨大的危险，敌人会通过这位联络人来针对你。 * 传说品质的联络人。 传说品质的联络人是拥有巨大权力和声望的人物，例如噬身之蛇的一名蛇之使徒，或埃雷波尼亚的一位高阶贵族。这些人物能以一己之力左右整个塞姆利亚大陆的事件。他们能获取最顶级的资源、政治力量，并知晓最危险的秘密。但是，向这样的联络人讨要人情可能会震动整个国家，并将你置于政治冲突的中心。

    铁道宪兵队（RMP） Heroic Government Authority 铁道宪兵队是埃雷波尼亚的军警混合机关，既是执法单位，也是与铁路网络绑定的机动部队。他们以精准高效而令人畏惧，与常规军队不同，他们的职权跨越军事与民间界限。 Rank 1. Cadet. 你每周挣得300米拉。配发英雄品质的铁道宪兵队装备，包括配枪、通讯装备与装甲制服。你有权拘留和调查军人及平民嫌疑人。 Rank 2. Officer. 你每周挣得500米拉。每次任务中，你可以申请额外的英雄品质装备，如卡宾枪或装甲车。你还可以召集一支3–5人的铁道宪兵队小队进行强制行动。 Rank 3. Captain. 你每周挣得800米拉。你被委派指挥分遣队（20–25名队员）。每次任务中，通过一次成功的声望检定，你可以征用史诗品质的资产，如监视无人机或机动炮。 Rank 4. Major. 你每周挣得1,200米拉。每次任务中，你可以征调史诗层级的铁道宪兵部队，例如整连（50人以上）或装甲列车。你的决定不仅影响任务，还塑造着埃雷波尼亚的公共秩序本身。 Rank 5. Commander. 你每周挣得1,800米拉。每次任务中，你可以两次调用铁道宪兵队的全部权限：动员装甲师、封锁整个铁路网络，或在帝国全境扣押资产。成功将获得全国赞誉，但失败也将终结职业生涯或危及生命。

    克洛斯贝尔特别支援科 英雄政府职权 特别支援科是克洛斯贝尔独特的实验性单位：一支被授权处理常规警力无法解决的案件的警察专案组，其行动方式常常超出传统手段。它的声誉建立在成员的创造力、主动性以及与公众的信任之上。成员身份带来自由，但也伴随着来自政界和地下世界的严密审视。 等阶 1. 初级搜查官。 你每周赚取300米拉。你将获得英雄品质的侦探装备，包括通讯器、法证工具和一把配发手枪。你被授予追查非常规案件的自由空间。 等阶 2. 搜查官。 你每周赚取500米拉。每次任务中一次，你可以申请普通品质的市政支援，例如借调的游击士资源、NPC盟友或市政设施。 等阶 3. 高级搜查官。 你每周赚取800米拉。你可以申请英雄品质的工具，例如监控车、战术背心或特种枪械。每次任务中一次，你可以通过一次声望检定，呼叫一名支援盟友（NPC侦探、游击士或市政人物）。 等阶 4. 主任搜查官。 你每周赚取1,200米拉。你拥有广泛的自由来调查任何案件。每次任务中一次，你可以申请史诗品质的装备，例如来自IBC的先进科技或财团的导力原型机。 等阶 5. 科长。 你每周赚取1,800米拉。每次任务中两次，你可以援引特别支援科的职权，赋予你警备队的临时指挥权、市政基础设施的完全准入权限，或者绕过政治限制的能力。你是克洛斯贝尔正义之战的门面，也是其敌人的目标。

    卡尔瓦德中央情报部 英雄政府职权 中央情报部（CID）是卡尔瓦德的情报机构，集谍报、内部安全和秘密行动于一体。他们因其无孔不入而令人畏惧，其特工都经过训练，能在高风险条件下独立行动。 等阶 1. 特工。 你每周赚取350米拉。你将获得英雄品质的秘密装备，包括消音手枪、开锁器和伪装工具包。你被授权在没有监管的情况下进行调查。 等阶 2. 外勤特工。 你每周赚取600米拉。每次任务中一次，你可以申请一个普通品质的卧底小组（3–4名特工）进行监视或渗透。 等阶 3. 高级特工。 你每周赚取950米拉。你可以申请英雄品质的装备，例如加密通讯器、先进枪械或监视无人机。每次任务中一次，你可以发起一次掩盖行动，抹除你行动的证据。 等阶 4. 部门主管。 你每周赚取1,400米拉。每次任务中一次，你可以申请史诗品质的情报资源，例如黑色预算、实验性原型机或精英突击队。 等阶 5. 局长圈。 你每周赚取2,000米拉。每次任务中两次，你可以援引CID职权来改变任务：冻结资产、勒索官员，或在卡尔瓦德任何地方部署突击部队。失败不是一个选项，共和国会埋葬它的错误。

    埃雷波尼亚情报局

    英雄级政府机关

    情报局是埃雷波尼亚的影子之手，负责谍报、黑色行动和政治监视。其特工以冷酷无情和高效率著称，常以帝国安全为名在法律监管之外行事。他们与铁血之子（Ironbloods）的区别在于，情报局效忠于埃雷波尼亚这一体制，而非奥斯本个人，尽管二者之间的界限时常模糊。

    Rank 1. 初级探员
    你每周赚取350米拉。你获得英雄品质的隐秘装备，包括随身武器、监听设备和加密文件。你有权调查民事与军事目标。

    Rank 2. 外勤探员
    你每周赚取600米拉。每次任务中，你可以征用一支普通品质的突击队（2至3名特工）执行监视或支援。你的身份在官方层面可被否认。

    Rank 3. 高级探员
    你每周赚取950米拉。你可以征用英雄品质的情报装备，如加密通讯、监视无人机或黑色行动武器。每次任务中，你可以授权一次行动掩盖，抹除你参与的证据。

    Rank 4. 部门主管
    你每周赚取1,400米拉。每次任务中，你可以征用史诗品质的资源，包括暗杀小队、高级原型武器或国家机密。你的职权堪比将军，但失败将招致铁血宰相的注意。

    Rank 5. 局长核心圈
    你每周赚取2,200米拉。每次任务中，你可以两次调用情报局的全部力量：冻结资产、拦截敌方行动或以伪装部署精锐营队。你是埃雷波尼亚最令人畏惧之人，也时刻处于对手的注视之下。

    利贝尔王室亲卫队

    英雄级政府机关

    王室亲卫队是宣誓守护利贝尔君主制的精锐部队，直接效忠于艾莉茜雅二世女王及其继承人。他们兼具礼仪与实战职能，是训练有素的士兵、驾驶员和王冠守护者。与常规军不同，他们的职权延伸至宫廷政治，使其成为捍卫王位与王国的双重重盾。

    Rank 1. 亲卫队学员
    你每周赚取300米拉。你获得英雄品质的制服和随身武器，并接受宫殿安保训练。你被委以基础礼仪职责和次要的岗哨轮值。

    Rank 2. 亲卫队军官
    你每周赚取500米拉。每次任务中，你可以征用普通品质的王室装备，如精确步枪或通讯器材。你将负责次要王室成员或国事活动的安保细节。

    Rank 3. 亲卫队骑士
    你每周赚取850米拉。你可以征用英雄品质的装备，包括强化装甲、战斗步枪和实验性导力随身武器。每次任务中，你可以召唤一支4至6人的亲卫兵小队进行支援。

    Rank 4. 王室冠军
    你每周赚取1,200米拉。每次任务中，你可以征用史诗品质的装备，如原型飞艇武器、机械化步兵或重型装甲。你可以担任女王本人的贴身护卫。

    Rank 5. 亲卫队长
    你每周赚取1,800米拉。每次任务中，你可以两次调用亲卫队权威：动员整个亲卫队、掌控王室资产，或在整片大陆范围内保护君主制。你深受王位信赖，但背叛将动摇利贝尔之本。

    Papal Guard Heroic Religious Authority Requirements: Rank 3 Errant Priest

    在阿尔泰利亚，七耀教会的中心，宗教权威与武术力量交汇。High Priests指导教义与治理，而Papal Guard执行他们的意志，保护High Basilica与Primate of 爱德丝。他们的职责融合仪式、政治以及对教会圣遗物的武装防御。

    Rank 1. Acolyte Guard. 你每周赚取250米拉。你获得英雄品质的法衣和一把圣化的武器（杖、剑或火器）。你负责仪式保护并协助上级神职人员。

    Rank 2. Confessor. 你每周赚取500米拉。每次任务一次，你可以申领普通品质的教会装备，如圣遗物、圣水或经文护符。你还可以主导小型的朝圣护送或大教堂巡逻。

    Rank 3. Papal Knight. 你每周赚取850米拉。你可以申领英雄品质的装备，如圣化武器、防护圣遗物或实验性奇术装置。每次任务一次，你可以召集3–5名papal guards或发起一次小型权威仪式。

    Rank 4. Cardinal Guard. 你每周赚取1,200米拉。每次任务一次，你可以申领史诗品质的资源，如圣遗物、奇迹仪式或精锐的templars小队。你的权威延伸到多个教区。

    Rank 5. Knight Cardinal. 你每周赚取1,800米拉。每次任务两次，你可以行使Papal Authority，动员整个Guard，获得拥有巨大力量的圣遗物，或发动横跨大陆的教会法令。你的权力兼具精神与世俗，但违命即为异端。
